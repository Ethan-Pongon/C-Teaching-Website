Lesson 2
Using variables in C

Create a new integer variable a and set it equal to 5. Next, create a new integer variable b and set it equal to 7.

After creating both variables a and b, create a new character variable c using valid C syntax. Defining a character variable in C can be done in the same way as creating an integer variable, where int is replaced with char, for instance, the example

char c = 3;

will create a character variable with value 3. Using this information, assign the char variable c the value of a + b in proper C syntax.

Try it!

Creating functions in C

Functions are an integral part of programming in C. Functions allow the developer to reuse and run the same lines of code under potentially different circumstances.

Creating a function starts with the return value. In C, functions can return a wide range of data types, from integers, characters, floats, doubles, and so on. For this example, we will be using the integer return type.

Starting this function off, we will write int as our return value, followed by a space.

Immediately following the function return type, we will give our function a name. Function names and variables in C must not have any spaces between characters. For instance, C will not accept add numbers as a function name. Instead, we can use camelCase or underscores when naming our functions or variables, like addNumbers or add_numbers . Of course camelCase and underscoring are just good practices, but you can name variables or functions however you like.

After giving your function a name, place a pair of matching parenthesis after your function name. There does not need to be a space between the function name and the parenthesis, but it is good to always pick one or the other and be consistent with it. This pair of parenthesis will eventually contain our parameters for our function, but for now leave the parenthesis blank.

So far your function should look like the following:

int addNumbers()

Next, add a space after the pair of parenthesis and add a pair of curly brackets. Within the curly brackets, press return/enter to add a newline. These curly brackets we have just added will contain our actual code inside of the function. Add a second newline for nicer formatting inside the curly brackets, and create a new variable equal to 7.

Following these steps, your code should look similar to the following example:

int addNumbers() {
  int a = 14;
}

IMPORTANT NOTE: Whenever using parenthesis (), curly brackets {}, or square brackes [], they must ALWAYS be in pairs. If an open bracket or parenthesis is unmatched, your code will fail to compile!

Because we assigned the function addNumbers to return an integer, we must return some integer in the function before it can be used. To return a value from inside the function, you must use the return keyword. If our addNumbers function returns the value of a, then we must add the statement in our function return a;.

The completed addNumbers function should look like the following:

int addNumbers() {
  int a = 14;
  return a;
}

Now that our addNumbers function is completed, we may call the function in the following manner:

addNumbers();

Now we may use this funtion call inside of other bits of code. For this example, if we chose to set an integer a to be equal to 14, we could run int a = addNumbers();, and that would essentially be the same as the statement int a = 14;.

Creating a variable in C

When creating a variable in C, you must specify the specific type of variable you would like to create. When you want to create a new integer variable, you can state int x = 7; where int specifically mentions the data type, x is the name of the variable, and 7 is the value assigned to x.

After every variable declaration in C, there must be a semicolon after the declaration.

Using this information, please create the variable a that contains an integer equal to the value 4 in proper C syntax.
